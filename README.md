# UrbanPulse: AI City Planning Digital Twin

**Team:** CodingKawKaw0299
**Track:** Healthy Cities
**Languange used:** Python

## Project Overview
UrbanPulse is a preventative city planning tool that allows government officials to simulate infrastructure changes (Bike Lanes, Parks, Flyovers) and instantly visualize their impact on **Public Health ROI** and **Community Stress**, powered by the **SEA-LION AI** model.

## Installation Instructions

1. **Unzip the project folder.**
2. **Open your terminal/command prompt** in this folder.
3. **Install the required libraries** by running:
   pip install -r requirements.txt

## How to run the app
Run the Streamlit Dashboard:

streamlit run app.py
